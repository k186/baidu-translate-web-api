### 如何使用

注册百度翻译帐号免费版
http://api.fanyi.baidu.com/

获取appid和秘钥
http://api.fanyi.baidu.com/manage/developer

修改config的appid 和 token配置为 你自己的配置

```shell
yarn run build
```
